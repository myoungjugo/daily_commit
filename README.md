# 커밋 가이드
---
1.  **메인 브랜치에 공부한 것(폴더명)/ 공부 세부내용(`.md` 파일)로 커밋.**
    - ex) add new file 에서 node.js/javascript_basic.md
    - 노션에서 작성한 것을 복붙 혹은 gpt 써서 마크다운 형식으로 바꾸기
    - 이론 공부 -> 해당 폴더의 README.md 파일에 작성
    - 실습 공부 -> 해당 폴더에 그냥 커밋

2.  **`README.md` 파일에 작성.**
    - 자신이 공부한 분야(제목) 
    - 공부한거 커밋한 링크(`저장소이름/브랜치이름/파일이름` 의 형태)
